Welcome to Arkacraft SMP

This epik bedrock server focuses more to building more than grinding.

There are 3 nations and 6 cities availble for the new players.

In this server you can go far away from spawn and claim that land

by using nether travel, there are a lot of availible portals to

unexplored land that you can also claim.

The builds on the server will be exported to a map and be put

on the map page.

Vending machines and currency


In Arkacraft Smp there is a special item called arkabux used for

currency. how to use it is to find a vending machine, throw the

arkabux into the hopper on the vending machine and it will

throw the item it is selling

To craft arkabux surround a cobblestone with paper for 1 arkabux

and surround an iron with paper for 10 arkabux.
